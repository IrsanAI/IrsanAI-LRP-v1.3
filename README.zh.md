## 概览
LRP 是一种结构化提示协议，将上下文、任务、约束和输出格式分离，并在执行前进行必需的共振检查。

## 核心组件
1. 上下文层
2. 任务层
3. 约束层
4. 输出格式

## 共振检查
- 简要复述已理解的上下文/任务/约束
- 仅在存在歧义时提问
- 达成一致后执行

## 快速开始
你可以通过 GitHub Pages 直接使用该工具： https://irsanai.github.io/LRP-v1.3/

```bash
python3 -m http.server 4173
```
打开： `http://127.0.0.1:4173/index.html`
