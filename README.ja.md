## 概要
LRP は、コンテキスト・タスク・制約・出力形式を分離し、実行前に必須のレゾナンスチェックを行う構造化プロンプトプロトコルです。

## 主要コンポーネント
1. コンテキスト層
2. タスク層
3. 制約層
4. 出力形式

## レゾナンスチェック
- 解釈したコンテキスト/タスク/制約を簡潔に復唱する
- 曖昧な場合のみ確認する
- 整合後に実行する
